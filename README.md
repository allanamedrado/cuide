# cuide

Após subir um novo codigo, crie uma tag e suba também.

Tutorial:

-veja quais são as tags que já existem:<br>
git tag<br>
exemplo de resultado: <br><br>

v1.0.0<br>
v1.0.1<br>
v1.0.2<br>
v1.0.3<br>

então a partir dai você cria na sequencia<br>
<br>
-git tag v1.0.4

-git push origin main v1.0.4
